# 博客管理系统

## 项目介绍

博客管理系统是一个基于SpringBoot实现的博客管理系统，前端使用了React框架，数据库使用MongoDB，redis，目前实现了用户管理、博客管理、分类管理、项目展示等功能。

## 如何运行

确保你已安装了MongoDB，Redis。修改`application.yml`中的MongoDB，Redis配置，然后运行`BlogApplication`即可。
